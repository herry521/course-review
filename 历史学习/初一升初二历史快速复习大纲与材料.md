# 初一升初二历史快速复习大纲与材料

适用范围：七年级中国历史上、下册常见内容，即中国古代史。若学校教材版本略有差异，可按老师发的目录微调，但核心主线基本一致。

## 一、怎么用这份资料

目标不是把课本重新读一遍，而是先搭框架，再补关键词，最后练材料题。

建议每天 40-60 分钟，按下面顺序复习：

1. 先看“朝代时间轴”，知道每一段历史处在什么位置。
2. 再看“分时期复习”，记住每一时期的政治、经济、民族关系、文化。
3. 最后用“高频问答”和“默写清单”检测。

孩子历史薄弱时，最容易犯的错是只背零散知识点。正确做法是：先记时间顺序，再记每个朝代的关键词，再练“原因、措施、影响、启示”的表达。

夏、商、西周、东周、秦、西汉、东汉、三国、西晋、东晋、南北朝、隋、唐、五代十国、辽、北宋、西夏、金、南宋、元、明、清。

## 四、分时期复习大纲

### 1. 史前时期：中国境内早期人类与文明起源

核心问题：人类怎样从采集狩猎走向农耕定居？

| 知识点 | 需要掌握的内容 |
|---|---|
| 元谋人 | 我国境内目前已确认的较早古人类之一 |
| 北京人 | 会使用火，过群居生活，使用打制石器 |
| 山顶洞人 | 掌握磨光、钻孔技术，有审美意识 |
| 半坡居民 | 黄河流域，种粟，住半地穴式房屋，制作彩陶 |
| 河姆渡居民 | 长江流域，种水稻，住干栏式房屋 |
| 炎帝、黄帝 | 被尊为中华民族的人文初祖 |
| 尧舜禹 | 禅让制；大禹治水 |

重点理解：

- 半坡和河姆渡说明我国原始农业在南北方都有发展。
- 南北民居不同，主要受自然环境影响。长江流域潮湿多雨，所以干栏式房屋更适合；黄河流域相对干燥寒冷，所以半地穴式房屋更适合。

### 2. 夏商周：早期国家与社会变革

核心问题：中国早期国家怎样形成？

| 朝代/制度 | 关键词 |
|---|---|
| 夏朝 | 我国历史上第一个王朝；启继承王位，世袭制代替禅让制 |
| 商朝 | 甲骨文；青铜器；司母戊鼎、四羊方尊 |
| 西周 | 分封制；宗法观念；礼乐制度 |
| 东周 | 分为春秋和战国，社会大变革时期 |

分封制：

- 目的：巩固周王朝统治。
- 对象：宗亲、功臣、先代贵族。
- 内容：周王授予诸侯土地和人民，诸侯向周王承担镇守疆土、交纳贡品、随从作战等义务。
- 作用：前期扩大统治范围，稳定统治秩序；后期诸侯势力增强，容易造成分裂割据。

甲骨文：

- 是中国已发现的古代文字中年代较早、体系较完整的文字。
- 我国有文字可考的历史从商朝开始。

### 3. 春秋战国：社会变革与思想活跃

核心问题：为什么春秋战国是大变革时代？

社会变化主线：

铁制农具和牛耕出现并推广，生产力提高，井田制逐渐瓦解，新兴地主阶级力量增强，各国变法图强，诸侯争霸和兼并战争频繁。

商鞅变法：

| 方面 | 主要措施 |
|---|---|
| 政治 | 建立县制，由国君直接派官吏治理 |
| 经济 | 废除井田制，承认土地私有；奖励耕织 |
| 军事 | 奖励军功，按军功授爵 |

意义：

- 使秦国国力大增，提高军队战斗力。
- 为秦统一全国奠定基础。

启示：

- 改革要顺应时代发展。
- 改革会触动旧势力利益，但能推动社会进步。

百家争鸣：

| 学派 | 代表人物 | 主要思想 |
|---|---|---|
| 儒家 | 孔子、孟子、荀子 | 仁、礼；仁政；重视教育 |
| 道家 | 老子、庄子 | 顺应自然；无为而治 |
| 墨家 | 墨子 | 兼爱、非攻、尚贤、节俭 |
| 法家 | 韩非 | 以法治国，建立中央集权 |
| 兵家 | 孙武 | 知己知彼，百战不殆 |

常考区别：

- 儒家重道德教化。
- 法家重法律和君主权威。
- 道家主张顺应自然、无为而治。

### 4. 秦汉：统一多民族国家的建立和巩固

核心问题：秦汉怎样建立并巩固大一统？

秦朝：

| 项目 | 内容 |
|---|---|
| 建立 | 公元前 221 年，秦王嬴政统一全国，建立秦朝 |
| 政治 | 创立皇帝制度；中央设丞相、太尉、御史大夫；地方实行郡县制 |
| 经济 | 统一货币、度量衡 |
| 文化 | 统一文字为小篆 |
| 交通 | 统一车辆和道路宽窄 |
| 军事 | 北击匈奴，修筑长城；开发南方 |
| 灭亡原因 | 秦的暴政，包括沉重赋税、徭役、刑罚和秦二世统治残暴 |

秦统一的意义：

- 结束长期分裂局面。
- 建立我国历史上第一个统一的多民族封建国家。
- 推动各地经济文化交流。

秦朝中央集权制度：

皇帝掌握最高权力；中央官员分管行政、军事、监察；地方实行郡县制，地方官由朝廷任免。这样有利于加强中央对地方的控制。

西汉：

| 知识点 | 内容 |
|---|---|
| 文景之治 | 休养生息，轻徭薄赋，社会经济恢复发展 |
| 汉武帝大一统 | 政治上推恩令；思想上罢黜百家、尊崇儒术；经济上盐铁专卖、统一铸币；军事上北击匈奴 |
| 张骞通西域 | 加强汉朝与西域联系，为丝绸之路开通创造条件 |
| 丝绸之路 | 促进东西方经济文化交流 |

汉武帝措施要会分类：

- 政治：推恩令，削弱诸侯王势力。
- 思想：把儒家学说立为正统思想。
- 经济：盐铁专卖、统一铸造五铢钱。
- 军事：派卫青、霍去病北击匈奴。

科技文化：

- 司马迁写《史记》，是我国第一部纪传体通史。
- 东汉蔡伦改进造纸术。
- 张仲景被称为“医圣”，著有《伤寒杂病论》。
- 华佗发明“麻沸散”，创编“五禽戏”。

### 5. 三国两晋南北朝：政权分立与民族交融

核心问题：分裂时期为什么也有进步？

三国形成：

- 官渡之战：曹操打败袁绍，为统一北方奠定基础。
- 赤壁之战：孙刘联军打败曹操，奠定三国鼎立局面。
- 三国：魏、蜀、吴。

江南地区开发：

| 原因 | 内容 |
|---|---|
| 北人南迁 | 带来劳动力、先进生产工具和技术 |
| 自然条件 | 江南自然条件优越 |
| 社会环境 | 南方相对安定 |
| 统治者重视 | 采取有利于农业发展的政策 |

影响：

- 江南经济迅速发展。
- 为后来经济重心南移奠定基础。

北魏孝文帝改革：

| 措施 | 作用 |
|---|---|
| 迁都洛阳 | 加强对中原地区统治，学习汉族先进文化 |
| 说汉语 | 促进民族交流 |
| 穿汉服、改汉姓 | 推动鲜卑族汉化 |
| 与汉族通婚 | 促进民族交融 |
| 学习汉族制度 | 推动北方社会发展 |

意义：

- 促进民族交融。
- 增强北魏实力。
- 推动北方社会经济文化发展。

科技文化：

- 祖冲之把圆周率精确到小数点后第七位。
- 贾思勰写《齐民要术》，是我国现存较早的一部完整农书。
- 王羲之被称为“书圣”，代表作《兰亭集序》。
- 云冈石窟、龙门石窟体现佛教艺术发展。

### 6. 隋唐：繁荣与开放的时代

核心问题：隋唐为什么能繁荣？又为什么由盛转衰？

隋朝：

| 知识点 | 内容 |
|---|---|
| 统一 | 隋灭陈，结束南北长期分裂 |
| 大运河 | 以洛阳为中心，北达涿郡，南至余杭，加强南北交通 |
| 科举制 | 隋炀帝时进士科创立，科举制正式形成 |
| 灭亡原因 | 隋炀帝暴政，大兴土木，连年征战，徭役沉重 |

大运河意义：

- 加强南北地区政治、经济、文化交流。
- 有利于国家统一和经济发展。

科举制意义：

- 打破门第限制，扩大官吏选拔范围。
- 促进社会阶层流动。
- 推动教育和文学发展。
- 加强皇帝选官用人权。

唐朝：

| 时期/人物 | 关键词 |
|---|---|
| 唐太宗 | 贞观之治；虚心纳谏；知人善任；轻徭薄赋 |
| 武则天 | 发展科举，重视人才，社会经济继续发展 |
| 唐玄宗前期 | 开元盛世，唐朝进入鼎盛时期 |
| 安史之乱 | 唐朝由盛转衰 |

唐朝民族关系：

- 唐太宗被北方和西北各族首领尊为“天可汗”。
- 文成公主入藏，促进唐蕃经济文化交流。

唐朝对外交流：

| 事件 | 方向 | 意义 |
|---|---|---|
| 遣唐使来华 | 日本到中国 | 学习唐朝制度和文化 |
| 鉴真东渡 | 中国到日本 | 传播佛教、医药、建筑等文化 |
| 玄奘西行 | 中国到天竺 | 促进中印佛教文化交流 |

唐朝特点：

- 政治较清明。
- 经济繁荣。
- 民族关系开明。
- 对外交流活跃。
- 文化开放自信。

### 7. 宋元：民族政权并立与经济重心南移

核心问题：宋朝军事不强，为什么经济文化发达？

北宋政治：

| 政策 | 内容与影响 |
|---|---|
| 重文轻武 | 防止武将专权和地方割据，但也导致军队战斗力削弱 |
| 加强中央集权 | 分化事权，削弱地方权力 |

民族政权并立：

| 政权 | 建立民族 | 重要关系 |
|---|---|---|
| 辽 | 契丹族 | 与北宋有战有和，澶渊之盟后较长时间和平 |
| 西夏 | 党项族 | 与北宋议和 |
| 金 | 女真族 | 灭辽、灭北宋；与南宋对峙 |
| 南宋 | 汉族政权 | 岳飞抗金；宋金和议后南北对峙 |

经济重心南移：

| 表现 | 内容 |
|---|---|
| 农业 | 占城稻推广；水稻产量提高；苏湖熟，天下足 |
| 手工业 | 南方丝织业、制瓷业发展；景德镇兴起 |
| 商业 | 城市商业繁荣；出现世界上较早的纸币“交子” |
| 海外贸易 | 广州、泉州等港口繁荣，设置市舶司管理海外贸易 |

完成时间：

- 经济重心南移到南宋时最终完成。

原因：

- 北方人口南迁，带来劳动力和技术。
- 南方自然条件较好。
- 南方战乱较少，社会相对安定。
- 统治者重视经济发展。

宋元文化科技：

- 北宋毕昇发明活字印刷术。
- 指南针广泛应用于航海。
- 火药武器广泛用于战争。
- 司马光主持编写《资治通鉴》，是编年体通史。
- 宋词代表人物有苏轼、李清照、辛弃疾。
- 元曲代表人物有关汉卿，代表作《窦娥冤》。

元朝：

| 知识点 | 内容 |
|---|---|
| 建立 | 忽必烈建立元朝，定都大都 |
| 统一 | 元朝结束长期分裂局面，完成全国统一 |
| 行省制度 | 中央设中书省，地方设行中书省，加强中央对地方管理 |
| 台湾 | 设置澎湖巡检司，管辖澎湖和琉球 |
| 西藏 | 设置宣政院，中央政府对西藏正式行使行政管辖 |
| 民族交融 | 形成新的民族回族 |

行省制度意义：

- 加强中央对地方的控制。
- 对后世地方行政制度影响深远。

### 8. 明清：统一多民族国家的巩固与危机

核心问题：明清怎样加强专制和巩固边疆？为什么也埋下危机？

明朝：

| 知识点 | 内容 |
|---|---|
| 建立 | 朱元璋建立明朝，定都应天府 |
| 强化皇权 | 废除丞相和中书省，权分六部；设厂卫特务机构 |
| 八股取士 | 束缚思想，不利于选拔真正有才能的人 |
| 郑和下西洋 | 最远到达非洲东海岸和红海沿岸 |
| 戚继光抗倭 | 基本解除东南沿海倭患 |
| 李自成起义 | 提出“均田免赋”口号，推翻明朝 |

郑和下西洋意义：

- 增进中国与亚非国家和地区的友好往来。
- 展示明朝国力。
- 是世界航海史上的壮举。

清朝：

| 方面 | 内容 |
|---|---|
| 加强皇权 | 雍正时设军机处，君主专制进一步强化 |
| 思想控制 | 大兴文字狱，压制思想文化 |
| 台湾 | 郑成功收复台湾；清朝设置台湾府，隶属福建省 |
| 西藏 | 册封达赖、班禅；设置驻藏大臣；实行金瓶掣签制度 |
| 新疆 | 平定大小和卓叛乱；设置伊犁将军 |
| 东北 | 抗击沙俄，签订《尼布楚条约》 |
| 闭关锁国 | 严格限制对外贸易，只开放广州一处对外通商较长时间 |

闭关锁国影响：

- 在一定程度上抵御了西方殖民侵略。
- 但限制中外交流，使中国逐渐落后于世界发展潮流。

明清文化：

- 明代李时珍写《本草纲目》。
- 宋应星写《天工开物》。
- 徐光启写《农政全书》。
- 明清小说繁荣，如《三国演义》《水浒传》《西游记》《红楼梦》。
- 京剧在清代逐渐形成。

## 五、专题复习表

### 专题 1：政治制度演变

| 制度 | 朝代 | 作用/特点 |
|---|---|---|
| 禅让制 | 尧舜禹时期 | 推举贤德之人担任部落联盟首领 |
| 世袭制 | 夏朝开始 | 王位在一家一姓中传承 |
| 分封制 | 西周 | 巩固统治，但后期容易导致诸侯割据 |
| 郡县制 | 秦朝 | 地方官由中央任免，加强中央集权 |
| 科举制 | 隋朝形成，唐朝完善 | 通过考试选官，扩大选官范围 |
| 三省六部制 | 隋唐 | 分工明确，提高行政效率，加强皇权 |
| 行省制度 | 元朝 | 加强中央对地方管理，对后世影响深远 |
| 废丞相 | 明朝 | 皇权进一步加强 |
| 军机处 | 清朝 | 君主专制达到高峰 |

### 专题 2：统一与分裂

| 阶段 | 特点 |
|---|---|
| 夏商周 | 早期国家产生和发展 |
| 春秋战国 | 诸侯争霸兼并，社会大变革 |
| 秦汉 | 大一统国家建立和巩固 |
| 三国两晋南北朝 | 政权分立，民族交融，江南开发 |
| 隋唐 | 重新统一，繁荣开放 |
| 五代辽宋夏金 | 民族政权并立 |
| 元明清 | 统一多民族国家进一步发展和巩固 |

结论：统一是中国历史发展的主流，分裂时期也会出现民族交融、经济开发和文化发展。

## 六、高频问答背诵卡

1. 为什么说春秋战国是社会大变革时期？

铁制农具和牛耕推广，生产力提高；井田制逐渐瓦解；新兴地主阶级兴起；各国变法图强；诸侯争霸和兼并战争推动统一趋势发展；思想上出现百家争鸣。

2. 商鞅变法为什么能使秦国强大？

政治上建立县制，加强国君对地方控制；经济上奖励耕织、承认土地私有，促进农业发展；军事上奖励军功，提高军队战斗力。因此秦国国力大增，为统一全国奠定基础。

3. 秦朝巩固统一采取了哪些措施？

政治上创立中央集权制度，地方实行郡县制；经济上统一货币和度量衡；文化上统一文字；交通上统一车辆和道路宽窄；军事上北击匈奴，修筑长城。

4. 秦朝灭亡的根本原因是什么？

秦朝实行暴政，赋税沉重，徭役繁重，刑罚严酷，秦二世统治更加残暴，激化社会矛盾，最终导致农民起义和秦朝灭亡。

5. 汉武帝怎样巩固大一统？

政治上实行推恩令，削弱诸侯王势力；思想上尊崇儒术；经济上实行盐铁专卖和统一铸币；军事上派兵北击匈奴。这些措施巩固了大一统局面。

6. 丝绸之路有什么作用？

丝绸之路促进了汉朝与西域以及东西方之间的经济文化交流，推动了商品、技术、文化的传播。

7. 江南地区为什么能得到开发？

北方人口南迁，带来劳动力和先进生产技术；南方自然条件优越；南方战乱较少，社会相对安定；统治者重视发展生产。

8. 北魏孝文帝改革有什么意义？

促进民族交融，增强北魏实力，推动北方社会经济文化发展，也为以后隋唐统一和多民族国家发展创造条件。

9. 科举制有什么影响？

扩大官吏选拔范围，使有才学的人能通过考试参政；促进社会阶层流动；推动教育和文学发展；加强皇帝选官用人权。

10. 唐朝繁荣开放表现在哪些方面？

政治上出现贞观之治和开元盛世；经济上农业、手工业、商业繁荣；民族关系上实行开明政策；对外交流上有遣唐使、鉴真东渡、玄奘西行；文化上诗歌等成就突出。

11. 宋代经济发展有什么表现？

农业上水稻产量提高；手工业上制瓷业、丝织业发达；商业上城市商业繁荣，出现纸币交子；海外贸易发达，设置市舶司管理贸易。南宋时经济重心南移完成。

12. 元朝行省制度有什么意义？

行省制度加强了中央对地方的管理，巩固了国家统一，对后世地方行政制度产生深远影响。

13. 明清君主专制怎样加强？

明朝废除丞相和中书省，权分六部，设厂卫特务机构，实行八股取士；清朝设军机处，大兴文字狱。君主专制不断强化。

14. 郑和下西洋的意义是什么？

郑和下西洋增进了中国与亚非国家和地区的友好往来，展示了明朝国力，是世界航海史上的壮举。

15. 闭关锁国有什么影响？

闭关锁国在一定程度上抵御了西方殖民侵略，但限制了中外经济文化交流，使中国逐渐落后于世界发展潮流。

## 七、材料题答题模板

### 1. 先看设问类型

| 设问词 | 答题方向 |
|---|---|
| “根据材料” | 答案主要从材料中概括，不能只背课本 |
| “结合所学” | 必须调用课本知识 |
| “原因” | 从政治、经济、民族、对外、人物、制度等角度找 |
| “措施” | 分条写具体做法 |
| “影响/意义” | 写积极作用，必要时写消极影响 |
| “启示” | 写今天能得到的道理，语言要概括 |

### 2. 历史材料题三步法

第一步：圈时间、人物、地点、制度名称。

第二步：判断属于哪个朝代和哪个专题。

第三步：按“材料信息 + 所学知识 + 结论”作答。

### 3. 常用答题句式

原因类：

- 政治上：国家统一、政局稳定、政策推动。
- 经济上：生产工具进步、劳动力增加、交通便利。
- 民族关系上：民族交往交流交融加强。
- 对外关系上：对外开放、交流频繁。

影响类：

- 有利于巩固统一。
- 有利于加强中央集权。
- 促进经济发展。
- 促进民族交融。
- 促进中外经济文化交流。
- 推动社会进步。

启示类：

- 改革要顺应时代潮流。
- 国家统一和社会稳定有利于经济发展。
- 开放交流促进进步，闭关自守容易落后。
- 民族交往交流交融推动统一多民族国家发展。

## 十一、家长陪复习的简单方法

每天只追问三个问题：

1. 今天复习的是哪个朝代？
2. 这个朝代最重要的 3 个关键词是什么？
3. 如果考试问“原因、措施、影响”，你能各说一句吗？

不要一开始就让孩子死背整段文字。先让孩子把关键词说出来，再补完整句子。历史考试尤其看重“关键词准确”和“表达分条”。

## 十二、升初二衔接提醒

初二上册通常会进入中国近代史。七年级最后复习明清时，要特别注意两个转折：

1. 明清前期统一多民族国家继续巩固，说明中国古代国家发展到较高水平。
2. 清朝后期闭关锁国、君主专制强化、思想文化受压制，使中国逐渐落后于世界潮流。

进入初二后，孩子会学到鸦片战争、中国开始沦为半殖民地半封建社会等内容。理解这部分时，可以从七年级明清的知识接上：

- 内因：清朝专制强化、闭关锁国、社会矛盾加剧。
- 外因：西方国家完成工业革命，对外殖民扩张。
- 结果：中国古代史结束，中国近代史开始。

所以七升八复习最后一定要把“开放促进进步，闭关容易落后”这条线记牢。
